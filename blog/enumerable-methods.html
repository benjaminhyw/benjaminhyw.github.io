The group_by enumerable method

Let's talk about the sort_by enumerable method!

What I like about the sort_by method is that, as the name suggests, there's a specific way we're about to sort our collection! What's nice is, we can make that whatever we want! Rather than automatically having it sort alphabetically or through index number, we can tell it to do whatever we want it to.

This is pretty useful when working with something like an array or hash, for example, and we want to sort the value any specific way. The basic notation looks like this:

a.sort_by {|variable1| variable.method}

Here we're working with an array and sorting with whatever method we implemented. That can be .to_s , .length, you name it!

Similarly when working with a hash, that notation might look like:

a.sort_by {|variable1, variable2| variable2.method}

In which case variable2 is the value, and we leave the key alone!
