Let's talk about the group_by enumerable method!

The ruby doc definition is : "Groups the collection by result of the block. Returns a hash where the keys are the evaluated result from the block and the values are arrays of elements in the collection that correspond to the key. If no block is given an enumerator is returned."

Basically, this method allows us to create groups in the form of hashes, depending on how we define the method! Our block defines the keys, and our value is the result!
